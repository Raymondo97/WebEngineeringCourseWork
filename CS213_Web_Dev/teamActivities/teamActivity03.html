Instructions

For all stylesheets, Jeffery, Claudia, and Jayden each participated with style and design change ideas and they worked together to get the CSS code to do what we wanted it to do.

Stylesheet 1 will provide a black background with background-color property. Each paragraph has a hover property to change the background of that paragraph to white. We also used the color property to change the font color. Each of the titles were given a different shadow effect with font-shadow properties. Using two font-shadow values for each title gave both a hard and soft shadow to each one. We used a fixed position for the menu bar. We applied a color property, a text-align property, a border property, a border-radius property, and a margin property for the menuBarItem id. For the menuBarItem class, we added a hover effect that changed the background to purple and the font size to 38 pixels. We also set a flex value for the document class's display property.

Stylesheet 2 changes the background to a specified rgba value with background-color property. We removed the styling form the list items in the menu bar. We set the menu bar to be aligned centered at the top of the page, and made them line up horizontally with the display property.We changed the alignment of the subject headings to a right justified alignment and centered all other text. We added a shadow to the hyperlinks and set the menu to change the background and text color on hover. All text size was modified, increasing the size and italicizing the subject headings. The fonts were changed for all text on the page as well.

Stylesheet 3 changes the background to a predefined color with the background-color property. The list that makes up the menu was modified to remove the bullet points, and was given a fixed position in the upper left corner of the page. We aligned all other text as centered, then gave it a padding to prevent it intersecting with the menu. This also made it more responsive to viewport size changes. We modified the cursor settings so that it will produce the poining hand cursor when hovering over the menu links. We changed the font size of the titles. We also changed the font of the paragraph content, as well as its size. We added a text shadow to the titles and menu. We removed the bullet points from the menu and the underlines form the hyperlinks. We changed the colors of all text as well.
